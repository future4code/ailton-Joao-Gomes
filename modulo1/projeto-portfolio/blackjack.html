Projeto BlackJack

O objetivo deste projeto foi criar um jogo semelhante a Blackjack (também conhecido como "21") utilizando Java Script. O programa recebe o usuário com um Confirm, onde o usúario escolhe entre iniciar uma nova rodada ou não, caso escolha iniciar o programa sorteia 2 cartas das 52 possibilidades (4 Naipes 13 cartas por Naipe) para o usuário e outras 2 para o computador, em seguida o programa mostra as cartas sorteadas via console.log, analisando a pontuação do usuário e do computador e finalmente indicando quem ganhou.
